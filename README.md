# React Geofence Demo

[View it here!](https://vv1141.github.io/react_geofence_demo/)

React Geofence Demo demonstrates usage of [React](https://react.dev/), and is written in [Typescript](https://www.typescriptlang.org/). It uses [Next.js](https://nextjs.org).

## Features

* Map and geofence rendering using [React Leaflet](https://react-leaflet.js.org/) and [OpenStreetMap](https://www.openstreetmap.org)
* Adding, removing, and dragging boundary markers

## Building from source

```bash
git clone https://github.com/vv1141/react_geofence_demo.git
cd react_geofence_demo
npm install
npm run build
```

## Running the development server

```bash
npm run dev
```
